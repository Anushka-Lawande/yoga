## 🛠️ Common Tasks

### Change Theme Color
Edit `src/index.css`, line with `--accent-primary`

### Add New Page
1. Create `src/pages/NewPage.jsx`
2. Create `src/pages/NewPage.css`
3. Add route to `App.jsx`

### Modify Components
Edit JSX files in `src/components/`
CSS files are in same folder

### Adjust Animations
Edit keyframes in `src/index.css`
Change `--transition-base` for speed
